- 🖼️ **مولد الصور بالذكاء الاصطناعي** - صور احترافية من وصف نصي
- ✍️ **كاتب النصوص بالذكاء الاصطناعي** - كتابة مقالات ومحتوى إبداعي
- 💬 **مساعد المحادثة الذكي** - إجابات فورية ومساعدة ذكية

## 🔒 ميزات الخصوصية المتقدمة

- 👻 **وضع الشبح (Ghost Mode)** - اختيار من يرى حالة الاتصال

- 🚫 **منع حذف الرسائل** - الاحتفاظ بجميع الرسائل
- 🚫 **منع حذف الحالات** - عدم اختفاء الحالات
- 👁️ **منع عرض مرة واحدة** - حفظ جميع الوسائط
- ✅ **علامة زرقاء بعد الرد** - تأكيد القراءة عند الرد فقط
- 🕐 **إخفاء وقت الحذف** - عدم إظهار وقت حذف الرسائل
- 📝 **إخفاء علامة التعديل** - عدم إظهار أن الرسالة معدلة

## 🔧 متطلبات البناء

- [apktool](https://ibotpeaches.github.io/Apktool/) - لفك وتجميع الـ APK
- Android SDK مع `build-tools` و `platforms`
- Java JDK 17+

```bash
# بناء APK من الكود المصدري
apktool b <اسم_المجلد> -o <اسم_النسخة>.apk

# توقيع الـ APK
apksigner sign --ks your-keystore.jks --out signed.apk unsigned.apk
```
