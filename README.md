- 🖼️ **مولد الصور بالذكاء الاصطناعي** - صور احترافية من وصف نصي

- 💬 **مساعد المحادثة الذكي** - إجابات فورية ومساعدة ذكية

- 💻 **مساعد البرمجة** - توليد وتصحيح الأكواد

- 🎵 **مولد الموسيقى** - تأليف مقطوعات موسيقية أصلية

## 🔧 متطلبات البناء

- [apktool](https://ibotpeaches.github.io/Apktool/) - لفك وتجميع الـ APK
- Android SDK مع `build-tools` و `platforms`
- Java JDK 17+

```bash
# بناء APK من الكود المصدري
apktool b <اسم_المجلد> -o <اسم_النسخة>.apk

# توقيع الـ APK
apksigner sign --ks your-keystore.jks --out signed.apk unsigned.apk
```
